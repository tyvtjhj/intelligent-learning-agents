## 1. 概念云地图
讲一个有多个部件的系统时**开场用**。先给骨架，再逐块填肉。

```
                    ╔═══════════════════╗
                    ║   ☁ 核心概念 ☁    ║
                    ╚═════════╤═════════╝
          ┌───────────────────┼───────────────────┐
          ▼                   ▼                   ▼
     ┌─────────┐         ┌─────────┐         ┌─────────┐
     │ 子概念A │ ──────▶ │ 子概念B │ ──────▶ │ 子概念C │
     └─────────┘         └─────────┘         └─────────┘
       挂的细节            挂的细节            挂的细节
```

## 2. 套娃 / 分层封装
任何"层层包裹/解包""一层套一层"的结构。

```
   ┌─────────────────────────────────────────────┐
   │ 最外层  [标签]      内容                       │
   │ 中间层  [标签] ┃    内容                       │
   │ 内层    [标签] ┃ ┃  内容                       │
   │ 核心    真正的数据                             │
   └─────────────────────────────────────────────┘
        正向 = 从外往里包；反向 = 从里往外拆
```

## 3. 术语账本表格
纪律 3 的标准载体。每次较长回复末尾更新。

```
| 简写 | 全称 | 一句话费曼 | 所属概念 |
|------|------|-----------|----------|
| 🆕 X | Full Name | 大白话一句 | 模块 |
```

## 4. 时序握手
两方来回交互(请求-确认、协商、握手)。

```
   甲 📱                                  乙 🖥
   │  ①「第一句:发起」                      │
   │ ─────────────────────────────────────▶│
   │  ②「第二句:回应 + 反问」               │
   │ ◀─────────────────────────────────────│
   │  ③「第三句:确认」                      │
   │ ─────────────────────────────────────▶│
   │ ════════ 此后进入正式状态 ════════      │
```

## 5. 信号 / 波形
物理层、频率、调制、穿透等。

```
低频/长波  ︵    ︵    ︵    ︵      绕得过障碍，但慢
          ╲╱  ╲╱  ╲╱  ╲╱

高频/短波  ∿∿∿∿∿∿∿∿∿∿∿∿∿∿      快，但一撞墙就被吸收
          ▓▓▓ 墙 ▓▓▓
```

```
████████████████████████████████
 ●→╲    ╱╲    ╱╲    ╱╲    ╱→
    ╲__╱  ╲__╱  ╲__╱  ╲__╱
████████████████████████████████
```

## 6. 木桶 / 瓶颈
对比多个量级，点明"最短板决定整体"。

```
  环节A  ║▓▓▓▓▓▓▓▓▓║ 866
  环节B  ║▓▓▓▓▓║     500
  环节C  ║▓▓║         120  ◄── 最短板!实际就卡这
  环节D  ║▓▓▓▓║       300
        ─────────────
  实际   ║▓▓║         ≈120
```

## 7. 分流 / 分支
决策、条件、"走A还是走B"。

```
            ┌─ 满足条件? ─┐
          是 │             │ 否
            ▼             ▼
       ┌─────────┐   ┌─────────┐
       │ 走 A 路 │   │ 走 B 路 │
       └─────────┘   └─────────┘
```

## 8. 接力转发
逐跳路径、链路、"一站站传递"。

```
起点 ●──▶[节点1]──▶[节点2]──▶ 🌊远距离 🌊 ──▶[节点N]──▶ ● 终点
        每跳只知道"往哪个方向更近"，不知道全程
```
